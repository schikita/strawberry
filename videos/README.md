# Видео для сайта

Поместите сюда файлы:
- **hero.mp4** — фон hero-секции
- **bg_cycle.mp4** — фон секции «Производственный цикл»
- **full-video.mp4** — основное видео
- **2.mp4** — видео «Шмели в помощь»

Папка `videos/*.mp4` исключена из Git (лимит 100 MB). **Перед загрузкой на сервер** запустите `copy-videos.ps1` в корне strawberry — скрипт скопирует видео из родительской папки. Либо загрузите папку `videos/` на сервер вручную (FTP и т.п.).
